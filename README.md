# Ticket System - IT Support

A simple but powerful C# console application for registering, listing, and managing IT support tickets. The system exports all data to CSV format, compatible with Excel and Google Sheets.

## Features

- Create new support tickets
- List all registered tickets
- Change ticket status (Open / In Progress / Resolved)
- Automatic timestamp
- CSV export for data analysis
- Persistent storage (data saved locally)

## 🚀 How to Run

### Prerequisites
- [.NET SDK](https://dotnet.microsoft.com/download) installed
- Or Visual Studio / VS Code

### Steps
1. Clone this repository
2. Open terminal in project folder
3. Run:
   ```bash
   dotnet run

## CSV Output
The program generates a tickets.csv file with the following columns:

Date and time
Requester name
Department
Problem description
Priority (High/Medium/Low)
Status (Open/In Progress/Resolved)

### You can import this file into:
Google Sheets
Microsoft Excel
Any spreadsheet software
